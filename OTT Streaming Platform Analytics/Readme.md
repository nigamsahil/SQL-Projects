# OTT Streaming Platform Analytics (MySQL Project)

This project analyzes user engagement, content performance, and subscription trends for an OTT streaming platform.

## Project Features
- User watch behavior analysis
- Content popularity & genre performance
- Subscription revenue & retention metrics
- Feedback and rating analysis

## Technologies Used
- MySQL / SQL
- Workbench / phpMyAdmin

## How to Use
1. Import the SQL file into MySQL Workbench
2. Run the schema and insert queries
3. Run provided analytics queries to generate insights
